# TachFileTo

**High Performance / Hybrid Rust-Python Evidence Loading System**

This project is designed to handle large file evidence loading with a defensive, tiered strategy.

## Architecture
- **Frontend**: React/TypeScript
- **Core**: Rust (Tauri)
- **Heavy Lifting**: Python (Docling)
